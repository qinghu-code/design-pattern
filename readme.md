### 二十三种设计模式

1. 创建型模式（5种）：
   * 单例模式
   * 原型模式
   * 建造者模式
   * 工厂方法模式（简单工厂、多工厂）
   * 抽象工厂模式
2. 结构型模式（7种）
   * 门面模式（外观模式）
   * 适配器模式
   * 桥接模式
   * 代理模式（静态代理、动态代理）
   * 装饰器模式
   * 组合模式
   * 享元模式
3. 行为型模式（11种）
   * 模板方法模式
   * 策略模式
   * 责任链模式
   * 状态模式
   * 命令模式
   * 备忘录模式
   * 观察者模式
   * 访问者模式
   * 中介者模式
   * 迭代子模式
   * 解释器模式
